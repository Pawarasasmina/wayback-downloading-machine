# wb-ui

A Node.js application for Wayback Machine UI restoration.

## Installation

1. Clone the repository.
2. Run `npm install` to install dependencies.

## Usage

Run the server with `npm start` or `node server.js`.

## Features

- Core restoration functionality in `wb_restore_core.js`.
- Server setup in `server.js`.
